**题目描述**

有 N 个格子从左到右排成一排。初始时，所有的格子里都没有方块。

给你 Q 次查询（query，即操作），请按顺序处理。每次查询是以下两种类型之一：

- `1 x`：在从左数第 x 个格子里放入 1 个方块。放入后，如果**每个**格子里都至少有 1 个方块，则从每个格子里各移除 1 个方块。
- `2 y`：输出至少包含 y 个方块的格子数量。

**数据限制**

- 1 <= N <= 3 *10^5
- 1 <= Q <= 3 * 10^5
- 1 <= x <= N
- 1 <= y <= 3 * 10^5
- 所有输入的值均为整数。
- 至少存在一次第二种类型（`2 y`）的查询。

**输入格式**

输入通过标准输入给出，格式如下：

N Q

query_1

query_2

\vdots

query_Q

每次查询 query_i (1 <= i <= Q) 为以下两种格式之一：

```
1 x
```

或

```
2 y
```

**输出格式**

设 K 为第二种类型查询的数量。输出 K 行。

第 i 行 (1 <= i <= K) 应包含对第 i 个第二种类型查询的答案。

**样例输入 1**

```
3 7
1 1
1 3
1 3
2 1
2 2
1 2
2 1
```

**样例输出 1**

```
2
1
1
```

**样例解释 1**

N=3，初始时从左数第 1、第 2、第 3 个格子里的方块数量分别为 (0,0,0)。查询按如下顺序处理：

1. 在从左数第 1 个格子里放入 1 个方块。此时存在没有方块的格子，因此不执行进一步操作。第 1、第 2、第 3 个格子里的方块数量变为 (1,0,0)。
2. 在从左数第 3 个格子里放入 1 个方块。第 1、第 2、第 3 个格子里的方块数量变为 (1,0,1)。
3. 在从左数第 3 个格子里放入 1 个方块。第 1、第 2、第 3 个格子里的方块数量变为 (1,0,2)。
4. 包含至少 1 个方块的格子是从左数第 1 个和第 3 个，共有 2 个这样的格子。因此，输出 2。
5. 包含至少 2 个方块的格子只有从左数第 3 个，共有 1 个这样的格子。因此，输出 1。
6. 在从左数第 2 个格子里放入 1 个方块。此时，每个格子都至少包含 1 个方块，因此从每个格子里各移除 1 个方块。第 1、第 2、第 3 个格子里的方块数量变为 (0,0,1)。
7. 包含至少 1 个方块的格子只有从左数第 3 个，共有 1 个这样的格子。因此，输出 1。

因此，依次输出 2、1、1，每行一个。

### 题解

首先我们设置一个k，为全局的最小的那个盒子，所以，所有盒子的值都大于等于这个k，初始的时候这个值为0。

但是，我们如果给定的查询是``1 x``那么，全局就需要减去1，从最后的实际来看，其实就是所有盒子的值为``ax - k``    ax为第x个盒子中的值，cj中的值是什么，是大于等于j值的数量。

就像一个队伍一样，任何一个到达j的值的ax值都会去c[j]哪里去报道，而且ax的值还是会继续增长，所以是大于等于ax的值。

对于查询来说查询大于y的值的数量，就是查询c这个数组中大于y+k值的数量（因为实际上y已经减去k了，但我们并没有进行这一步操作）

代码如下，这日本人的代码写的也是十分的优美

```c++
#include <bits/stdc++.h>
using namespace std;

#define N (int)3e+5

int main(void){
	int a[N+1]={};
	int c[N+1]={};
	int n,q,t,x,mn=0;

	cin>>n>>q;
	for(int i=0;i<q;i++){
		cin>>t>>x;
		if(t==1){
			a[x]++;
			c[a[x]]++;
			if(c[a[x]]==n)mn=a[x];
		}
		if(t==2){
			if(x+mn>q)cout<<0<<endl;
			else cout<<c[x+mn]<<endl;
		}
	}

	return 0;
}

```

# 排序&二分

### 快速排序（模板）

给定你一个长度为 n 的整数数列。

请你使用快速排序对这个数列按照从小到大进行排序。

并将排好序的数列按顺序输出。

#### 输入格式

输入共两行，第一行包含整数 n。

第二行包含 n 个整数（所有整数均在 1∼109 范围内），表示整个数列。

#### 输出格式

输出共一行，包含 n 个整数，表示排好序的数列。

#### 数据范围

1≤n≤100000

#### 输入样例：

```
5
3 1 2 4 5
```

#### 输出样例：

```
1 2 3 4 5
```

#### 思考

1，随机初始化分界点（l，r，mid）。

2，将l-mid中大于w[mid]的数移动到mid-r。

3，通过递归处理左右两端，直到递归细化到两数比较，排序结束。

对于思考2，我们可以建立两个数组a，b然后把比w[mid]小的数放在a数组里面，比w[mid]大的数放在b数组里面，等到遍历完w数组之后，把a，b数组放回w数组。

```
#include <bits/stdc++.h>

//快速排序

using namespace std;

const int N = 1e6 + 10;

int n;
int q[N];

void quick_sort(int *q,int l,int r) {
	if (l >= r)return;//剪枝
	int x = q[l], i = l - 1, j = r + 1;
	while (i < j) {
		do i++; while (q[i] < x);
		do j--; while (q[j] > x);
		if (i < j)swap(q[i], q[j]);
	}
	quick_sort(q, l, j);
	quick_sort(q, j + 1, r);
}

int main(){
	scanf("%d", &n);
	for (int i = 1; i <= n; i++) {
		scanf("%d", &q[i]);
	}
	quick_sort(q,1,n);
	for (int i = 1; i <= n; i++) {
		cout << q[i] << " ";
	}
	return 0;
}

```

### 归并排序（模板）

给定你一个长度为 n 的整数数列。

请你使用归并排序对这个数列按照从小到大进行排序。

并将排好序的数列按顺序输出。

#### 输入格式

输入共两行，第一行包含整数 n。

第二行包含 n 个整数（所有整数均在 1∼109 范围内），表示整个数列。

#### 输出格式

输出共一行，包含 n 个整数，表示排好序的数列。

#### 数据范围

1≤n≤100000

#### 输入样例：

```
5
3 1 2 4 5
```

#### 输出样例：

```
1 2 3 4 5
```

#### 思考

1，递归排序左边和右边。

2，把两个数组合并。

难点在合并数组，用两指针，分别指向两个数组，放入指针指向值较小的那个数，然后该指针往后移动。直到遍历完其中一个数组，把剩下那个数组的值直接补到最后边。

可能有的人，会有疑问，为什么在归并排序的时候，把q当成两有序的数组，进行归并。因为我们第一步是将归并排序递归到最底部也就是，此时这个数组只有一个数，两个数自然是可以比较大小的。然后返回的时候我们就拥有两个长度为2的有序数组，依次类推直到数组的大小为n/2。

```
#include <bits/stdc++.h>

//归并排序

using namespace std;

const int N = 1e6 + 10;

int n;
int q[N],k,tmp[N];

void merge_sort(int *q,int l,int r) {

	if (l >= r)return;

	int mid = l + r >> 1;//  ">>1"的意思是除以2，并且向下取整

	merge_sort(q, l, mid),merge_sort(q, mid + 1, r);

	int k = 0, i = l, j = mid + 1;
	while (i <= mid && j <= r) {
		if (q[i] <= q[j])tmp[k++] = q[i++];
		else tmp[k++] = q[j++];
	}

	while(i <= mid)tmp[k++] = q[i++];
	while (j <= r)tmp[k++] = q[j++];

	for (int i = l, j = 0; i <= r; i++, j++)q[i] = tmp[j];
}

int main(){
	cin >> n;
	for (int i = 0; i < n; i++) {
		scanf("%d", &q[i]);
	}
	merge_sort(q, 0, n - 1);
	for (int i = 0; i < n; i++)cout << q[i] << " ";
	return 0;
}
```

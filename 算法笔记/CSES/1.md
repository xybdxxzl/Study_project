## Two Sets

你的任务是将数字 '1,2,...n 分成两个和相等的集合。

**输入**

唯一的一行输入包含一个整数 $n$。

**输出**

如果可以划分，打印 "YES"；否则打印 "NO"。

如果可以划分，接下来打印一个构造这两个集合的示例。首先，打印第一个集合中的元素个数，随后在新的一行打印这些元素本身；接着，以同样的方式打印第二个集合。如果有多种分法，输出任意一种即可。

**数据范围**

​	``1<=n<=1e6``

**示例 1**

**输入:**

7

**输出:**

YES

4

1 2 4 7

3

3 5 6

**示例 2**

**输入:**

6

**输出:**

NO

**思路**

这个题是首先考虑总和是不是被2整除，第二考虑怎么凑出总和的二分之一，直接从大往小凑，如果比剩余小，就放入第一个集合，比当前大就放入第二个集合，数学理论支持：从1到n可以表达出从1到（n）*（n+1）/2的所有数字。

## Gray Code

格雷码（Gray code）是一个包含所有 2^n 个长度为 n 的比特（位）串的列表，其中任意两个相邻的字符串恰好只有一位不同（即它们的汉明距离为 1）。

你的任务是为给定的长度 n 构造一个格雷码序列。

**输入**

唯一的一行输入包含一个整数 n。

**输出**

打印 2^n 行来描述格雷码。你可以打印任何有效的解。

**数据范围**

1 <= n <= 16

**示例**

**输入:**

2

**输出:**

00

01

11

10

```C++
#include<bits/stdc++.h>

using namespace std;

vector<string> g(int n) {
	if (n == 1) {
		return { "0","1" };
	}
	vector<string> prev = g(n - 1);
	vector<string> res;
	for (int i = 0; i < prev.size(); i++) {
		res.push_back("0" + prev[i]);
	}
	for (int i = prev.size()-1; i >= 0; i--) {
		res.push_back("1" + prev[i]);
	}
	return res;
}

int main() {
	int n;
	cin >> n;
	vector<string> ga = g(n);
	for (int i = 0; i < ga.size(); i++)cout << ga[i] << endl;
	return 0;
}
```

## Maximum Subarray Sum

Given an array of **n** integers, your task is to find the maximum sum of values in a contiguous, nonempty subarray.

**Input**

The first input line has an integer **n**: the size of the array.

The second line has n integers x1,x2,…,xn the array values.

**Output**

Print one integer: the maximum subarray sum.

**Constraints**

​	``	1≤n≤2⋅10^5``

​	``−10^9≤xi≤10^9``

**Example**

**Input:**

```
8
-1 3 -2 5 3 -5 2 2
```

**Output:**

```
9
```

**思路**

这个题是最长字段和问题，这个问题我一直的解决方式就是前缀和，利用一个非常新的写法。

对于连续的数字来说，如果前面的和是负数，其实就已经没有存在的必要了，因为这样只会拖后腿。

但是如果所有的数字都是负的，如此思考就会选成空集。

所以我们对于第**i**个数字的操作是，对比a[i],s[i-1]+a[i]即使全是负的，也存在一个最大的值。

```C++
for(int i=0;i<n;i++){
	cin>>x;//第i个数字
    cur=max(cur+x,x);//cur是当前最优的情况，即选到x后最优的情况
    ans=max(ans,cur);//和全局最大值ans比较，选出全局最大值

}
```



## String Reorder

Your task is to reorder the characters of a string so that no two adjacent characters are the same. What is the lexicographically minimal such string?

**Input**

The only line has a string of length **n** consisting of characters A–Z.

**Output**

Print the lexicographically minimal reordered string where no two adjacent characters are the same. If it is not possible to create such a string, print −1-1−1.

**Constraints**

​	``1≤n≤10^6``

**Example**

Input:

```
HATTIVATTI
```

Output:

```
AHATITITVT
```

**思路**

用cnt数组存储下每个字符的出现次数，用一个for循环遍历每个位置，这个位置可以放该字符的条件为1，当前字符与上次选择的字符不同。2，选择该字符后剩余字符的最大值不超过未选字符的一半。

```C++
for(int pos =0;pos<len;pos ++){
	bool found=0;//判断这个位置有没有的选
	for(int c=0;c<26;c++){
		if(last==c)continue;
		if(cnt[c]<=0)continue;
		
		cnt[c]--;
		
		int max1=-1;
		for(int i=0;i<26;i++){
			max1=max(max1,cnt[i]);
		}
		if(max1>(len-pos)/2){
			cnt[c]++;
		}
        else{
            ans.push_back(c);
            last=c;
            found=1;
            break;
		}
	}
    if(!found){
        //有一个位置无法被选中
        cout<<"-1";
        return 0;
    }
}
```

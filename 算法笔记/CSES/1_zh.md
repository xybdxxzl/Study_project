## Two Sets

你的任务是将数字 '1,2,...n 分成两个和相等的集合。

**输入**

唯一的一行输入包含一个整数 **n**。

**输出**

如果可以划分，打印 "YES"；否则打印 "NO"。

如果可以划分，接下来打印一个构造这两个集合的示例。首先，打印第一个集合中的元素个数，随后在新的一行打印这些元素本身；接着，以同样的方式打印第二个集合。如果有多种分法，输出任意一种即可。

**数据范围**

​	``1<=n<=1e6``

**示例 1**

**输入:**

7

**输出:**

YES

4

1 2 4 7

3

3 5 6

**示例 2**

**输入:**

6

**输出:**

NO

**思路**

这个题是首先考虑总和是不是被2整除，第二考虑怎么凑出总和的二分之一，直接从大往小凑，如果比剩余小，就放入第一个集合，比当前大就放入第二个集合，数学理论支持：从1到n可以表达出从1到（n）*（n+1）/2的所有数字。

## Gray Code

格雷码（Gray code）是一个包含所有 2^n 个长度为 n 的比特（位）串的列表，其中任意两个相邻的字符串恰好只有一位不同（即它们的汉明距离为 1）。

你的任务是为给定的长度 n 构造一个格雷码序列。

**输入**

唯一的一行输入包含一个整数 n。

**输出**

打印 2^n 行来描述格雷码。你可以打印任何有效的解。

**数据范围**

``1 <= n <= 16``

**示例**

**输入:**

2

**输出:**

00

01

11

10

```C++
#include<bits/stdc++.h>

using namespace std;

vector<string> g(int n) {
	if (n == 1) {
		return { "0","1" };
	}
	vector<string> prev = g(n - 1);
	vector<string> res;
	for (int i = 0; i < prev.size(); i++) {
		res.push_back("0" + prev[i]);
	}
	for (int i = prev.size()-1; i >= 0; i--) {
		res.push_back("1" + prev[i]);
	}
	return res;
}

int main() {
	int n;
	cin >> n;
	vector<string> ga = g(n);
	for (int i = 0; i < ga.size(); i++)cout << ga[i] << endl;
	return 0;
}
```

## Maximum Subarray Sum

给定一个包含 **n** 个整数的数组，你的任务是找出一个连续且非空的子数组中数值的最大和。

**输入**

第一行输入包含一个整数 **n**：数组的大小。

第二行包含 n 个整数 x1,x2,…,xn：数组的值。

**输出**

打印一个整数：最大子数组和。

**数据范围**

​	``	1≤n≤2⋅10^5``

​	``−10^9≤xi≤10^9``

**示例**

**输入:**

```
8
-1 3 -2 5 3 -5 2 2
```

**输出:**

```
9
```

**思路**

这个题是最长字段和问题，这个问题我一直的解决方式就是前缀和，利用一个非常新的写法。

对于连续的数字来说，如果前面的和是负数，其实就已经没有存在的必要了，因为这样只会拖后腿。

但是如果所有的数字都是负的，如此思考就会选成空集。

所以我们对于第**i**个数字的操作是，对比a[i],s[i-1]+a[i]即使全是负的，也存在一个最大的值。

```C++
for(int i=0;i<n;i++){
	cin>>x;//第i个数字
    cur=max(cur+x,x);//cur是当前最优的情况，即选到x后最优的情况
    ans=max(ans,cur);//和全局最大值ans比较，选出全局最大值

}
```





## String Reorder

你的任务是重新排列一个字符串的字符，使得没有两个相邻字符相同。满足该条件的字典序最小的字符串是什么？

**输入**

唯一的一行包含一个长度为 **n** 的字符串，由字符 A–Z 组成。

**输出**

打印字典序最小的重排字符串，其中没有两个相邻字符相同。如果无法创建这样的字符串，则打印 −1-1−1。

**数据范围**

​	``1≤n≤10^6``

**示例**

输入:

```
HATTIVATTI
```

输出:

```
AHATITITVT
```

**思路**

用cnt数组存储下每个字符的出现次数，用一个for循环遍历每个位置，这个位置可以放该字符的条件为1，当前字符与上次选择的字符不同。2，选择该字符后剩余字符的最大值不超过未选字符的一半。

```C++
for(int pos =0;pos<len;pos ++){
	bool found=0;//判断这个位置有没有的选
	for(int c=0;c<26;c++){
		if(last==c)continue;
		if(cnt[c]<=0)continue;
		
		cnt[c]--;
		
		int max1=-1;
		for(int i=0;i<26;i++){
			max1=max(max1,cnt[i]);
		}
		if(max1>(len-pos)/2){
			cnt[c]++;
		}
        else{
            ans.push_back(c);
            last=c;
            found=1;
            break;
		}
	}
    if(!found){
        //有一个位置无法被选中
        cout<<"-1";
        return 0;
    }
}
```

## Missing Coin Sum

你有 nnn 个正整数值的硬币。使用这些硬币的子集，你无法创建的最小和是多少？

**输入**

第一行有一个整数 nnn：硬币的数量。

第二行有 nnn 个整数 x1,x2,…,xnx_1,x_2,\dots,x_nx1,x2,…,xn：每个硬币的值。

**输出**

打印一个整数：最小的无法表示的硬币和。

**数据范围**

​	**1≤n≤2⋅10^5**

​	**1≤xi≤10^9**

示例

输入:

```
5
2 9 1 2 7
```

输出:

```
6
```

我定义一个sum表示这个数组能够表示出从1~sum中间的所有的数字。

sum的初始值是0，表示现在表示不了任何数字。

开始遍历a数组，那么如果用上a[i]，那我们可以表示的区间范围是**[** a[i] , sum+a[i] **]**

I）a[i]>=sum+1， 那么我们就可以续上这个区间，这个区间就是无缝衔接的

II）a[i]>sum+1，那么sum+1就永远无法表示出来，因为第i个元素之后的一但使用就会大于sum+1，而i个元素之前，最多只能表示出sum大小的元素

## Playlist

你获得了一个广播电台自成立以来的播放列表。该播放列表共有 **n** 首歌。

最长的连续歌曲序列是多少，其中每首歌都是唯一的？

**输入**

第一行输入包含一个整数 nnn：歌曲的数量。

下一行有 nnn 个整数 k1,k2,…,kn：每首歌的 ID 编号。

**输出**

打印最长的不重复歌曲序列的长度。

**数据范围**

​	**1≤n≤2⋅10^5**

​	**1≤ki≤10^9**

**示例**

输入:

```
8
1 2 1 3 2 7 4 2
```

输出:

```
5
```

在一个区间[l , r]如果两段都包含就是``r-l+1``,如果是左闭右开，或者左开右闭，那就是``r-l``，

对于这个题，我们固定一个滑动窗口[l,r]，定义一个map用来观察当前的k是不是曾经出现过，但是注意，即使是出现过，也有可能不在l~r的区间之内。所以我们更新l是max（l，mp[k]+1）要么是在这个区间之内，我们把所有重复之前的都踢出队列，要么是队列之外的，我们不用管。
